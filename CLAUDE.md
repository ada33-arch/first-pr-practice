# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Overview

This is a small practice repository for learning the GitHub workflow: cloning, branching, committing, and opening a pull request. It contains no application code, build system, or test suite — just a README.

## Working in This Repo

- There are no build, lint, or test commands. Changes are documentation/content edits verified by reading them.
- The default branch is `main`. Following the repo's purpose, changes should be made on a feature branch and proposed via pull request rather than committed directly to `main`.
